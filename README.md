# iGEM Alginate Film Analysis App (For Non-Developers)

이 문서는 개발 경험이 없는 사용자들이 앱을 설치하고 실행하는 방법을 상세하게 안내하기 위해 작성되었습니다.

## 📱 앱 소개

이 앱은 iGEM 프로젝트의 일환으로, 알지네이트 필름(Alginate Film)의 색 변화를 분석하여 중금속 검출 여부와 반응 면적을 계산하는 모바일 애플리케이션입니다.

**주요 기능:**

1. **필름 직경 설정**: 실제 필름의 크기(mm)를 입력하여 분석의 기준을 잡습니다.
2. **카메라 촬영**: 필름을 촬영합니다.
3. **이미지 분석**: 촬영된 이미지에서 전체 필름 영역과 반응이 일어난 구역(Spot)을 지정하면 자동으로 직경과 넓이를 계산합니다.

## 🛠 준비 사항 (설치 가이드)

앱을 실행하기 위해서는 컴퓨터에 몇 가지 프로그램이 필요합니다. 아래 순서대로 천천히 설치해주세요.

### 1단계: Node.js 설치

이 앱은 자바스크립트 기반으로 만들어졌기 때문에 Node.js라는 프로그램이 필요합니다.

1. [Node.js 공식 홈페이지](https://nodejs.org/ko/)에 접속합니다.
2. **LTS 버전** (안정적 버전, 예: 20.x.x LTS)을 다운로드하고 설치합니다.
3. 설치 중 나오는 설정은 모두 기본값(Next 클릭)으로 진행해도 괜찮습니다.

### 2단계: Git 설치

소스코드를 컴퓨터로 다운로드 받기 위해 Git이 필요합니다.

1. [Git 공식 홈페이지](https://git-scm.com/downloads)에 접속합니다.
2. 사용 중인 운영체제(Windows 또는 Mac)에 맞는 버전을 다운로드하고 설치합니다.
   - **Windows 사용자:** 설치 중 "Git Bash Here" 옵션이 체크되어 있는지 확인해주세요.
   - **Mac 사용자:** 터미널에서 `git --version`을 입력했을 때 설치하라는 팝업이 뜨면 설치를 진행합니다.

### 3단계: 휴대폰에 Expo Go 앱 설치

개발 중인 앱을 휴대폰에서 실행해보기 위해 필요합니다.

- **iPhone (iOS):** App Store에서 'Expo Go' 검색 및 설치
- **Android:** Play Store에서 'Expo Go' 검색 및 설치

## 🚀 앱 설치 및 실행하기

이제 준비가 끝났으니 앱을 실행해봅시다.

### 1. 소스코드 다운로드 (Clone)

1. **터미널(Mac)** 또는 **PowerShell/Git Bash(Windows)** 을 엽니다.
2. 코드를 저장하고 싶은 폴더로 이동합니다. (예: `cd Desktop`)
3. 아래 명령어를 복사해서 붙여넣고 엔터를 칩니다.

   ```bash
   git clone https://github.com/hhhkdev/igem-alginate-film-mobile.git
   ```

   _(참고: 주소는 실제 저장소 주소로 변경해주세요. 만약 파일로 받았다면 압축을 풀고 그 폴더로 이동하면 됩니다.)_

4. 다운로드된 폴더로 이동합니다.
   ```bash
   cd igem-alginate-film-mobile
   ```

### 2. 라이브러리 설치

앱 실행에 필요한 도구들을 설치하는 과정입니다. 아래 명령어를 입력하세요.

```bash
npm install
```

_(시간이 조금 걸릴 수 있습니다. 에러가 나지 않고 완료될 때까지 기다려주세요.)_

### 3. 앱 실행

설치가 완료되었다면 앱을 실행합니다.

```bash
npx expo start
```

명령어를 입력하면 화면에 **QR 코드**가 나타납니다.

### 4. 내 폰에서 확인하기

- **iPhone:** 기본 카메라 앱을 켜서 화면의 QR 코드를 스캔하세요. 'Expo Go에서 열기'를 누르면 앱이 실행됩니다.
- **Android:** Expo Go 앱을 켜고 'Scan QR Code'를 눌러 화면의 코드를 찍으세요.

⚠️ **주의사항:** 컴퓨터와 휴대폰이 **같은 와이파이(Wi-Fi)** 에 연결되어 있어야 합니다.

## 💡 면적 계산 원리 (Algorithm)

이 앱이 어떻게 사진만으로 실제 면적을 알아내는지 궁금하시죠? 그 원리는 **'비례식'** 입니다.

### 1단계: 기준 잡기 (Calibration)

앱 시작 시 사용자가 입력한 **'실제 필름 직경(mm)'** 이 기준이 됩니다.

- 예를 들어, 사용자가 필름 지름을 `25mm`라고 입력했다고 가정합니다.
- 사진 분석 화면에서 **파란색 원**으로 필름 전체를 맞춥니다.
- 이때 앱은 **"화면상의 파란 원 픽셀 크기 = 실제 25mm"** 라는 사실을 알게 됩니다.

### 2단계: 비율 계산 (Pixel to mm Ratio)

위 정보를 바탕으로 **1픽셀이 실제로 몇 mm인지** 계산합니다.

> **비율 = 실제 직경 (mm) / 파란 원의 지름 (pixel)**

### 3단계: 반응 구역 측정

사용자가 **빨간색 원**으로 색이 변한 구역(Spot)을 맞춥니다.
앱은 빨간 원의 픽셀 지름에 아까 구한 비율을 곱해서 실제 지름을 구합니다.

> **반응 구역 실제 지름 = 빨간 원 지름 (pixel) × 비율**

### 4단계: 최종 면적 계산

구해진 지름을 이용해 원의 넓이 공식($A = \pi r^2$)으로 면적을 계산합니다.

> **반응 면적 ($mm^2$) = $3.14 \times (\text{반응 구역 실제 지름} / 2)^2$**

## ❓ 문제 해결 (FAQ)

**Q. QR 코드를 찍었는데 앱이 안 열려요.**

- 컴퓨터와 휴대폰이 같은 와이파이에 연결되어 있는지 확인해주세요.
- 방화벽 문제일 수 있습니다. `npx expo start --tunnel` 명령어로 다시 시도해보세요.

**Q. 카메라 권한 허용 창이 안 떠요.**

- 휴대폰 설정 > 알지네이트 앱(또는 Expo Go) > 권한 > 카메라 허용으로 직접 설정해주세요.
